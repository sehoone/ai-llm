# 02. 데이터 타입 (Data Type)

파이썬의 기본 자료형과 주요 연산을 학습합니다.

## 학습 내용

| 주제 | 내용 |
|------|------|
| 기본 타입 | `int`, `str`, `float`, `bool` |
| 문자열 | f-string, `.format()`, 반복 연산 |
| 타입 변환 | `int()`, `str()`, `float()` |
| 리스트 | 인덱싱, 슬라이싱, `append`, `insert`, `remove`, `del` |
| 튜플 | 불변(immutable) 리스트 |

## 실행

```bash
# 의존성 설치 및 가상환경 초기화
uv sync

# 실행
uv run python main.py
```
